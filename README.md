# TextFlow

TextFlow is a simple command-line automation tool written in Rust.
It processes text files and outputs formatted or transformed text.

This project was created as part of a personal automation portfolio assignment.

---

## Features
- Command-line interface (CLI)
- Reads text from input files
- Processes and formats text
- Fast and lightweight

---

## Installation

Make sure Rust and Cargo are installed.

Clone the repository and build the project:

```bash
cargo build --release
